# Skills

Focused skills for repeatable `oh-my-copilot` workflows.

| Skill | Description | Bundled Assets |
| --- | --- | --- |
| `copilot-highlights-flush` | Reset local `.copilot-highlights` runtime artifacts safely. | `scripts/flush-highlights.cjs` |
| `copilot-highlights-recap` | Summarize the latest captured Highlights session into a concise report. | None |
| `copilot-highlights-init-deep` | Bootstrap instruction surfaces and starter plugin conventions. | None |
| `copilot-highlights-implementation-plan` | Turn a feature request into a PDCA-style implementation plan. | None |
| `copilot-highlights-verification-loop` | Verify plugin changes with fixture tests, install checks, and smoke tests. | `scripts/smoke-test-models.cjs` |

Notes:

- Skills are folders containing `SKILL.md` and optional bundled assets.
- Bundled scripts are referenced directly in the skill instructions.
